MY PROJECTS

Following are the projects undertaken by me:

  • STUDENT REPORT CARD MANAGEMENT SYSTEM:
    This project was designed by me in 2019 after completion of my course in C++ programming. This project included the concepts of OOPS and file handling.

  • DIABETES PREDICTION SYSTEM USING MACHINE LEARNING IN PYTHON:
    This project was designed by me in 2021 as a group project, which was a part of our curriculum in 6th Semester. This project was designed using Scikit-Learn library in Python language. This system works by comparing four separate machine learning algorithms, namely Linear Regression, Gaussian Naive Bayes, Random Forest and KNN. It then computes the best performing model, and uses it to compute the risk of diabetes for the user as per the data input given by the user.

  • TIME TABLE MANAGEMENT SYSTEM:
    This project was designed by me in 2021 as a group project, which was a part of our curriculum in 6th semester. This project implements the concepts of web development using HTML, CSS, and PHP. The entire system was linked with a SQL database, completely developed over Wamp server. The system took input of registered members, and displayed their time tables. Additionally, it had the functionality of adding new classes to the schedule, for faculty users of the system.

  • GREEN SCREEN IMPLEMENTATION:
    This project was designed by me in 2021 while studying web development from Coursera. This system uses the Canvas element to take input of an image, having a green screen as its background. It then replaces all green pixels with the pixels of the image, which will act as the background of the previous image. Thus it works to implement the green screen algorithm on images.

  • IMAGE FILTER APPLICATION SYSTEM:
    This project was designed by me in 2021 while studying web development from Coursera. This system uses the Canvas element to take input of an image, and then works on the pixels of the image to apply various filters on the image, namely Grayscale, Red Filter, and Rainbow filter.

  • MEASUREMENT UNIT CONVERSION SYSTEM:
    This project was designed by me in 2021 while studying PHP. This is designed using PHP language and works to convert various measurement units.
